---
name: feedback-dev-server-restart
description: Always restart dev server after changes
type: feedback
---

Always restart `npm run dev` after any file change. Stop the current server and start it fresh so changes are reflected immediately.
**Why:** User wants to verify changes in the browser right away, no manual restart needed.
**How to apply:** After any Edit/Write that modifies a component, stop the dev server with Ctrl+C and restart it before reporting completion.
